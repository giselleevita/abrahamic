# Brand assets

The `giselleevita/abrahamic` repo ships **no real logo, illustration, or icon
sprite** today. Its `public/` folder contains only the default Next.js
scaffolding SVGs and a single 4.9 MB hero video.

The marks under `assets/` are synthesised by this design system to match the
existing visual voice (warm dark canvas + Crimson Text wordmark + three
tradition dots). **Treat them as placeholders pending brand-owner approval.**

| File              | Purpose                                              |
| ----------------- | ---------------------------------------------------- |
| `logo-wordmark.svg` | Horizontal wordmark, light-on-dark — header use.   |
| `logo-mark.svg`     | Three-dot monogram in canonical tradition colors.  |
| `logo-footer.svg`   | Expanded "Abrahamic Texts" footer wordmark in gold.|
